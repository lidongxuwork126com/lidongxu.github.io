# Promise概念

  1. 它是异步操作的一个解决方案, 简单说它就是一个异步操作的容器, 在未来某一个时刻会完成的操作
  2. Promise是一个构造函数, 它身上有resolve和reject方法, 原型上有then和catch方法
  3. 它只有三种状态, pending(运行中) fulfilled(成功), rejected(失败)
  4. then() 接收resolve / catch() 接收reject / finally 无论如何都执行

# Promise.all()

用于把多个Promise对象, 合并成一个, 如何有一个失败, 直接进入catch, 如果子Promise自己实现了catch, 则合并的Promise对象不会进入catch, 进入then
Promise.all([p1, p2, p3]);

# Promise.resolve()

用于将对象转成Promise对象

Promise.resolve('foo') // 返回一个Promise对象
// 等价于
new Promise(resolve => resolve('foo'))

# 手写Promise实现

  1. Promise雏形, 先理解这个
// 声明Promise, fn就是你在new Promise传入的方法体
function Promise(fn) {
    var value = null;
    var callbacks = [];  //callbacks为数组，因为可能同时有很多个回调
    this.then = function (onFulfilled) { // 接收外面.then时传入进来的回调函数
        // 加入到callbacks数组中
        callbacks.push(onFulfilled);
        return this; // 返回当前Promise对象, 为了链式写法
    };
    function resolve(value) { // 当promise内部调用resolve方法时, 将.then传入进来的回调函数全部执行一遍
        callbacks.forEach(function (callback) {
            callback(value); // 把value值带出去, 带到then注册的方法参数中
        });
    }
    fn(resolve); // 调用new Promise时, 传入的方法体, 开始执行了
}

  1. 调用promise时, 发现resolve方法马上同步执行了, 所以还没等then调用呢.
function getUserId() {
    return new Promise(function (resolve) {
        resolve(9876);
    });
}
// 调用方法, 得到Promise对象. 调用then注册回调函数
getUserId().then(function (id) {
    console.log(id); // 我们发现什么也没有打印, 证明resolve先执行了, 而then还没调用.所以这里没有回调成功
});

  1. 如果失败的话怎么处理呢? (最终版) 加入状态判断
function Promise(fn) {
    var value = null; // resolve/reject的消息
    var callbacks = []; // 装载回调函数
    this.then = function (onFulfilled) { // 接收外面.then时传入进来的回调函数
        // 加入到callbacks数组中 (注意这里分成功, 失败情况)
        callbacks.push({
            "onFulfilled": onFulfilled
        });
        return this;
    };
    this.catch = function (onRejected) { // 接收外面.catch时传入进来的回调函数
        // 加入到callbacks数组中 (注意这里分成功, 失败情况)
        callbacks.push({
            "onRejected": onRejected
        });   
        return this;
    };
    // 当promise内部调用resolve方法时, 将.then传入进来的回调函数全部执行一遍
    function resolve(value) {
        execute("onFulfilled", value);
    }
    function reject(value) {
        execute("onRejected", value);
    }
    function execute(key, value) {
        setTimeout(function () {
            callbacks.forEach(function (callbackObj) {
                callbackObj[key] && callbackObj[key](value); // 把value值带出去, 带到then注册的方法参数中
            });
        }, 0);
    }
    fn(resolve, reject); // 调用new Promise时, 传入的方法体, 开始执行了
}

function getUserId() {
    return new Promise(function (resolve, reject) {
        resolve(9876);
        setTimeout(function () {
            // reject("失败");
        }, 2000);
    });
}

getUserId().then(function (id) {
    console.log(id);
}).catch(function (err) {
    console.log(err);
})
