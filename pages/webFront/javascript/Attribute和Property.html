  1. attribute -> HTML标签上的属性

<input checked="checked" value="值">

# 注意

注意: 标签上对attribute进行赋值，会同时赋给dom的property同名属性 注意: attribute和property之间的数据绑定是单向的，attribute->property 注意: 给property添加属性值, 不会同步给标签上的attribute 注意: 更改property和attribute上的任意值，都会将更新反映到网页的状态中
